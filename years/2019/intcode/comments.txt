Day 7 program (examples)
a = Get computer id
b = Previous output
b *= 10 // Shift output digits to the right
a += b // Add current id
output a

a = Get computer id
b = Previous output
b *= 10 // Shift output digits to the right
a *= -1 // Invert a
a += 5 // Add 5 (a = 5 - computer id)
a += b // Add current id
output a

a = Get computer id
b = Previous output
b *= 10 // Shift output digits to the right
a -= 2 // Subtract 2 from a
if (a < 0) a += 7 // Add 7 if a is negative (a = 5 if id = 0, 6 if id = 1, others are a = id - 2)
a += b // Add current id
output a
